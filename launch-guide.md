## PART 2: GOOGLE SEARCH CONSOLE — STEP BY STEP

### Step 1: Go to Search Console
Open https://search.google.com/search-console/
Sign in with the Google account you want to manage the site with.

### Step 2: Add Property
Click "Add property" (top-left dropdown).
Select "URL prefix" (easier than Domain verification for GitHub Pages).
Enter: `https://baselinepsychotherapy.com/`
Click "Continue."

### Step 3: Verify Ownership
Google will offer multiple verification methods. The easiest for GitHub Pages:

**Option A — HTML file (recommended):**
1. Google gives you a file like `google1234567890abcdef.html`
2. Download it
3. Upload it to your GitHub repo (same process as the other files — "Add file" → "Upload files")
4. Commit to main
5. Wait 1-2 minutes for GitHub Pages to deploy
6. Go back to Search Console and click "Verify"

**Option B — HTML tag:**
1. Google gives you a meta tag like `<meta name="google-site-verification" content="abc123...">`
2. Open your index.html
3. Add the tag inside `<head>`, after the charset meta tag
4. Upload the updated index.html to GitHub
5. Go back to Search Console and click "Verify"

### Step 4: Submit Sitemap
Once verified:
1. In the left sidebar, click "Sitemaps"
2. In the "Add a new sitemap" field, type: `sitemap.xml`
3. Click "Submit"
4. Status should show "Success" (may take a few minutes)

### Step 5: Request Indexing
1. In the left sidebar, click "URL Inspection"
2. In the top search bar, paste: `https://baselinepsychotherapy.com/`
3. Click Enter
4. If it says "URL is not on Google," click "Request Indexing"
5. Repeat for `https://baselinepsychotherapy.com/policies.html`

### What Happens Next
- Google typically crawls and indexes within 2-7 days after submission
- You can check progress by searching `site:baselinepsychotherapy.com` on Google
- Once indexed, the FAQ schema may start showing rich results within 1-2 weeks
